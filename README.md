# Sistem Informasi Manajemen Stok Barang (Inventory System)

Aplikasi berbasis web untuk mengelola inventaris barang secara digital. Sistem ini dirancang untuk mempermudah pencatatan stok, proses stok opname, hingga pembuatan laporan yang efisien.

* **Stok Opname**: Fitur sinkronisasi stok fisik dengan data sistem untuk meminimalisir selisih.
* **Scan Barang**: Fitur pencarian atau input barang melalui skrip scan.
* **Export Laporan**: Fitur mencetak atau ekspor data laporan stok ke format Microsoft Excel.
* **Responsive Design**: Antarmuka yang rapi menggunakan CSS dan JavaScript khusus.
